# (Go/Gin/Gorm) ToDoアプリ
## 1. 概要
- 現在、Go/Gin/GORMを学習していて、学習の一環で作成しました。
## 2. 使用技術
### 言語
- Go: `1.19.3`
### フレームワーク
- Gin: `1.8.1`
### データベース
- MySQL: `5.7`
### インフラ
- Docker: `20.10.21`
- docker-compse: `2.12.2`

## 3. 起動方法
- 開発環境に`Docker`を使用しているので、事前に`Docker`をインストールしてください。
- 起動するには`.env`が必要です。
```
$ docker-compose up -d
```
